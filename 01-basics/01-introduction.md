# Go Fiber পরিচিতি

## Fiber কী?

**Fiber** হলো Go (Golang) প্রোগ্রামিং ভাষার জন্য একটি আধুনিক, দ্রুত এবং শক্তিশালী ওয়েব ফ্রেমওয়ার্ক। এটি Node.js এর জনপ্রিয় **Express.js** ফ্রেমওয়ার্ক থেকে অনুপ্রাণিত।

## কেন Fiber ব্যবহার করবেন?

### ১. অত্যন্ত দ্রুত (Blazingly Fast)
Fiber **fasthttp** এর উপর নির্মিত, যা Go-এর সবচেয়ে দ্রুত HTTP ইঞ্জিন। এটি স্ট্যান্ডার্ড `net/http` প্যাকেজের চেয়ে **10 গুণ** পর্যন্ত দ্রুত হতে পারে।

### ২. Express-এর মতো সহজ API
```go
// Express.js (JavaScript)
app.get('/user/:name', (req, res) => {
    res.send('Hello ' + req.params.name)
})

// Fiber (Go) - প্রায় একই রকম!
app.Get("/user/:name", func(c fiber.Ctx) error {
    return c.SendString("Hello " + c.Params("name"))
})
```

### ৩. শূন্য মেমোরি অ্যালোকেশন
Fiber রিকোয়েস্ট হ্যান্ডলিংয়ে **zero-allocation** নীতি অনুসরণ করে, যা মেমোরি ব্যবহার কমায়।

### ৪. মিডলওয়্যার সাপোর্ট
রিকোয়েস্ট প্রসেস করার আগে/পরে কোড চালানোর সুবিধা।

### ৫. রোবাস্ট রাউটিং
ডায়নামিক রাউট, রাউট গ্রুপিং, প্রিফিক্স ইত্যাদি সাপোর্ট করে।

## প্রয়োজনীয়তা

Fiber ব্যবহার করতে আপনার প্রয়োজন:

1. **Go 1.21** বা তার উপরের ভার্সন
2. যেকোনো কোড এডিটর (VS Code রিকমেন্ডেড)
3. টার্মিনাল/কমান্ড প্রম্পট

## প্রথম ধারণা

Fiber অ্যাপ্লিকেশনের মূল উপাদান:

```go
package main

import "github.com/gofiber/fiber/v3"

func main() {
    // অ্যাপ তৈরি করুন
    app := fiber.New()
    
    // রাউট ডিফাইন করুন
    app.Get("/", func(c fiber.Ctx) error {
        return c.SendString("Hello, World!")
    })
    
    // সার্ভার শুরু করুন
    app.Listen(":3000")
}
```

### বিশ্লেষণ:

1. **`fiber.New()`** - নতুন Fiber অ্যাপ্লিকেশন তৈরি করে
2. **`app.Get()`** - GET রিকোয়েস্টের জন্য রাউট ডিফাইন করে
3. **`fiber.Ctx`** - রিকোয়েস্ট ও রেসপন্স হ্যান্ডল করার কনটেক্সট
4. **`app.Listen()`** - নির্দিষ্ট পোর্টে সার্ভার শুরু করে
